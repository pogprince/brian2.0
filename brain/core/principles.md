# Principles

- Reduce friction from idea to execution
- Memory is append-only by default — never destructively delete
- Agents should be disposable until proven reusable
- Context should load automatically, not require manual setup
- The system should remain inspectable and transparent
- Flow manages writeback, not individual workers
- Multi-project support from day one
