# Constraints

- Local-first: no external database required
- Markdown as source of truth
- TypeScript + Next.js stack
- Claude as default LLM provider
- No hardcoded business-specific agents
- Memory is never destructively deleted
- Writeback is flow-managed, not arbitrary
