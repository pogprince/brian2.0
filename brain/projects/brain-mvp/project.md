# Brain MVP

## Description
A modular agent operating system with markdown-based memory, flow-state orchestration, and a control-plane dashboard.

## Stack
- Next.js + React + TypeScript
- Tailwind CSS
- Markdown file storage
- Claude API (default LLM)

## Status
Active development — building initial MVP.
